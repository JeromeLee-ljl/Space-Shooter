# Space-Shooter
弹幕类游戏——第一个unity游戏

### 操作
wasd ：移动

鼠标左键 : 开火

1: 普通子弹

2: 霰弹

3: 激光

4：引导弹 （击中目标后一定时间内可以引导普通子弹和霰弹追踪敌机）

q：导弹 （击中目标后造成范围伤害）

e：护盾 （抵挡普通子弹）

space：闪现 （向鼠标位置闪现一段距离）
